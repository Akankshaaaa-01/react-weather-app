# React Weather App 🌦️

A simple weather application built using **React + Vite** that fetches real-time weather data using the **OpenWeather API**.

## Features

* Search weather by city
* Displays temperature, humidity, and wind speed
* Weather description and icon
* Error handling for invalid cities

## Tech Stack

* React
* Vite
* Material UI
* OpenWeather API

## Installation

Clone the repository:

git clone https://github.com/Akankshaaaa-01/react-weather-app.git

Install dependencies:

npm install

Run the project:

npm run dev

## Environment Variables

Create a `.env` file in the root folder and add:

VITE_WEATHER_API_KEY=your_api_key_here
